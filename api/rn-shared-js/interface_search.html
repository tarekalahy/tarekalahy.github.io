Object Search::search ( Location  startLoc,
Location  endLoc,
integer  modifiers,
DOMString  textToFind 
)

Searches for ALL succesful results for the specified string between the start and end locations. Note: If the search is backwards or the document is rtl, then the start location will be closer to the end of the document than the end location The returned object is the results.

results is an array of the form:

[
{ start:loc, end:loc, result:string },
{ start:loc, end:loc, result:string },
{ start:loc, end:loc, result:string }
etc.
]
Parameters
startLocif null, startLoc is the beginning of the document
endLocif null, endLoc is the end of the document.
modifiersbitwise combination of flags (see above)
findTextthe text to find
Returns
an array of the results
