Object SpineItem::getAttributes ( )

Returns an array of objects that is the complete set of attributes for this item, where the array is of the form:

[
{ ns:string, name:string, value:string },
{ ns:string, name:string, value:string },
{ ns:string, name:string, value:string }
etc.
]

where
ns is the fully qualified namespace
name is the short name of the attribute (without prefix or namespace)
value is the value of the attribute

Returns
an array of objects that hold the complete set of attributes
