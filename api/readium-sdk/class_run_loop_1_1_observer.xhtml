enum RunLoop::Observer::ActivityFlags : uint32_t

The different Activity flags.

Enumerator
RunLoopEntry 

Observe entry to the run loop.

RunLoopBeforeTimers 

Fires before any timers are processed.

RunLoopBeforeSources 

Fires before any event sources are processed.

RunLoopBeforeWaiting 

Fires before the RunLoop waits for events/timers to fire.

RunLoopAfterWaiting 

Fires when the RunLoop finishes waiting (event/timer/timeout).

RunLoopExit 

Fired when RunLoop::Run(bool, std::chrono::duration) is about to return.
