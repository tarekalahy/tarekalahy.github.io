Methods

alternateSpread

() String

Returns the inversion of the spineItem's SPREAD property. i.e if the page-spread is right it returns LEFT and vice versa. If the spread is center then it returns CENTER

Returns:

String:

getRenditionFlow

() String

Returns a string indicating the type of layout for viewport overflow, i.e. scrolldoc, scroll-continuous, paginated or auto. Note that if the spineItem has an override (local value) that is returned, else the package's value is returned

Returns:

String:

(

Checks to see if the specified property is set in this spineItem and matches the supplied value. If the property is NOT set in the spineItem then the the package is checked. If not set in either place then the function returns FALSE.

Properties

href

String

The href of the spine item, i.e. the URI to the resource in the EPUB which the spineitem will render

Default: None

idref

String

The idref of the spine item, i.e. the ID-based pointer to the actual manifest item that the spine item references

Default: None

index

String

The index of this spine item in the spine itself.

Default: None

linear

String

A flag indicating whether the spineItem has the attribute linear, which is either yes or no. Default is yes.

Default: yes

media_overlay_id

String

The ID, if any, of the root SMIL element of the media overlay for the document.

Default: None

media_type

String

The mimetype of this specific spine item.

Default: None

page_spread

String

A variable indicating the type of synthetic spread for this specific spine item, where page:spread-* can be left, right or center or auto

Default: auto

rendition_flow

String

A string specifying how "overflow" content that exceeds the current viewport should be laid out. Possible values are paginated, scrolled-continuous, scrolled-doc or auto

Default: auto
